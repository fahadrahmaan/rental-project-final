# Rental Portal 🚗🏍️🏡

A full-stack rental management system built with **Python Django, HTML, CSS, and SQLite**.  
The platform allows service providers to list cars, bikes, and homestays, while customers can browse, book, and pay seamlessly.  
It includes an intelligent **recommendation system** that suggests rentals based on user activity and booking history.  

## User Roles
- **Admin** – Manage users, services, payments, and overall operations.  
- **Service Providers** – Add and manage cars, bikes, and homestays.  
- **Customers** – Explore services, make bookings, and complete payments.  

## Features
- Service listings (cars, bikes, homestays)  
- Booking requests & payment management  
- Dashboards for admin & providers  
- Notifications & records  
- Recommendation system powered by user history  
- Secure authentication & role-based access  

## Tech Stack
- **Backend**: Python Django  
- **Frontend**: HTML, CSS  
- **Database**: SQLite
